
#  Smart Calculator Website

A modern, responsive Smart Calculator web application built with a clean UI and smooth user experience. This project performs basic arithmetic operations with real-time input handling and a visually appealing design.
The calculator is designed to be simple, intuitive, and beginner-friendly while maintaining a polished, professional look suitable for portfolio projects.

#  Features

1. Basic arithmetic operations: Addition, Subtraction, Multiplication, Division.
2. Percentage (%) calculation
3. Clear (C) and Backspace functionality
4. Real-time expression display
5. Modern UI with gradient background and soft shadows
6. Responsive layout (works well on different screen sizes)
